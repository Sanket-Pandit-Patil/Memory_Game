# 🎮 Brainy Flip: A Premium Memory Challenge

**Brainy Flip** is a high-fidelity, interactive memory game built with React and Vite. Experience a sophisticated user interface combined with challenging gameplay mechanics, including time constraints, limited moves, and dynamic difficulty settings.

---

## ✨ Features

### 💎 Premium UI/UX
- **Glassmorphism Design**: Sleek, translucent cards with high-fidelity shadows and backdrop filters.
- **Dynamic Theming**: Seamless transition between sophisticated Light and Dark modes.
- **Micro-Animations**: Smooth entry animations, hover scaling, and celebratory confetti effects.
- **Responsive Layout**: Optimized for a perfect single-page fit on various viewport sizes, from mobile to desktop.

### 🧠 Gameplay Mechanics
- **3 Difficulty Levels**: Choose from Easy, Medium, and Hard presets.
- **Custom Challenges**: Set your own limits! Challenge yourself with:
    - **Time Limits**: Racing against the clock.
    - **Move Constraints**: Planning every flip carefully.
- **Real-time Stats**: Track your moves, time, and match progress during the game.
- **Win/Fail Summary**: Detailed performance report including accuracy and best scores.

### 🔊 Immersive Audio
- **Responsive Feedback**: Auditory cues for tile flips and successful matches.
- **Clean Experience**: Selective sound logic that emphasizes victories while keeping mismatches silent and focused.

---

## 🛠️ Technical Stack

- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **State Management**: React Hooks (useState, useEffect, useMemo, useCallback, useRef)
- **Styling**: Vanilla CSS with modern Flexbox/Grid and Design Tokens
- **Icons**: [Lucide React](https://lucide.dev/)
- **Effects**: [Canvas-confetti](https://www.npmjs.com/package/canvas-confetti)
- **API**: Custom emoji fetch logic for dynamic board generation

## 🚀 Getting Started

### Prerequisites
- Node.js (v16.0.0 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Sanket-Pandit-Patil/Memory_Game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd memory_game
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Build
Generate a production-ready build:
```bash
npm run build
```

## 🎨 Design Philosophy

Brainy Flip follows a **"Soft UI"** and **"Glassmorphism"** aesthetic. Every element is carefully defined with:
- **Layered Shadows**: For deep, realistic elevation.
- **Pronounced Borders**: Subtly defined dark borders for professional structure.
- **Vibrant Gradients**: Carefully curated HSL-tailored colors for brand identity.
